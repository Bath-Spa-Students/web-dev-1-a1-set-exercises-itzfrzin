HTML & CSS Quick Reference Guide

HTML Tags

HTML tags are used to structure content on a webpage.

<div>

The <div> tag is a block-level container used to group content or elements and apply styles.

<div>
    Content here
</div>

<p>

The <p> tag defines a paragraph.

<p>This is a paragraph</p>

<img>

The <img> tag is used to embed images.

<img src="image.jpg" alt="Description">

<a>

The <a> tag defines a hyperlink.

<a href="https://example.com">Link</a>

CSS Properties

CSS properties are used to style HTML elements.

color

Sets the text color.

color: #333;

font-family

Sets the font family for text.

font-family: Arial, sans-serif;

background-color

Sets the background color.

background-color: #fff;

margin

Sets the margin around an element.

margin: 10px;

padding

Sets the padding inside an element.

padding: 10px;

border

Sets the border of an element.

border: 1px solid #ccc;

text-decoration

Sets text decoration, like underline or line-through.

text-decoration: none;

display

Sets the display behavior of an element.

display: block;
